# ZWeArt Marketing Site

Modern, conversion-focused marketing site for ZWeArt - a Toronto-based art school for kids and teens ages 6-17.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Fonts: Sora (headings) + Inter (body)

## Getting Started

### Using pnpm (recommended)

```bash
pnpm install
pnpm dev
```

### Using npm

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` - Home page
- `/programs` - Programs overview
- `/contact` - Contact form

## Project Structure

```
zweart/
├── app/              # Next.js App Router pages
├── components/
│   ├── ui/           # Reusable UI components
│   └── sections/     # Page sections
└── public/           # Static assets
```
